# Alfred工作流-有道词典

功能基于有道词典官网，**无需任何配置**即可实现Alfred快速**翻译**与**发音**。

## 功能

- [x] 中英文翻译
  - 单词翻译 Enter键 进行复制
- [x] 支持单词发音
  - ⌥ Option键 + Enter键 英式发音
  - ⌘ Command键 + Enter键 美式发音


## 安装

下载workflow文件安装后即可使用，无需任何条件（使用Js开发，使用Nodejs进行执行，已经内置Nodejs可执行文件）。第一次运行的时候需要进行授权：
![164211638971514_ pic](https://user-images.githubusercontent.com/27878293/145219760-439159f2-3854-4a78-923d-163d4fe92421.jpg)

**授权操作** 点击屏幕左上角  → 系统偏好设置 → 安全性与隐私 → 通用 点击允许按钮进行授权（只需操作一次）。
